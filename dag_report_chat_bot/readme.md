Python, Airflow, bot Telegram

В телеграмм чат


### Что в отчете

* Краткая справка по сновным метрика(и сравнение с предыдущим днем и неделей назад)
* Графики

### Файлы

1. Черновик отчета (личные данные скрыты)
2. Файл .py. для дага Airflow (личные данные скрыты)
